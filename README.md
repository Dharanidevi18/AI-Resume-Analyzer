# AI Resume Analyzer

## Overview

AI Resume Analyzer is a Java-based ATS (Applicant Tracking System) Resume Analyzer that evaluates resumes against job descriptions. It extracts skills from resumes, compares them with job requirements, calculates a match percentage, identifies missing skills, and provides improvement recommendations.



## Features

- Resume PDF Parsing using Apache PDFBox  
- Automatic Skill Extraction  
- Job Description Analysis  
- Skill Matching Engine  
- ATS-Style Job Match Calculation  
- Missing Skill Detection  
- Resume Improvement Suggestions  
- Learning Recommendations for Missing Skills  



## Tech Stack

- Java  
- Object-Oriented Programming (OOP)  
- Apache PDFBox  
- Java Collections Framework  
- Rule-Based Matching Logic  
- NLP-Style Text Processing  



## Project Workflow

1. Upload or provide a resume PDF  
2. Extract text using Apache PDFBox  
3. Identify skills from resume  
4. Parse job description  
5. Extract required skills  
6. Compare resume skills with job requirements  
7. Calculate match percentage  
8. Generate missing skill report  
9. Provide improvement suggestions
