# AI Resume Analyzer

A Java-based ATS Resume Analyzer that extracts skills from resumes, matches them with job descriptions, calculates ATS scores, and provides recommendations for skill improvement.

## Features
- Resume PDF Parsing
- Skill Extraction
- ATS Score Calculation
- Job Description Analysis
- Skill Matching
- Missing Skill Detection
- Recommendation Engine

## Tech Stack
- Java
- Apache PDFBox
- Maven
- Collections Framework
- NLP-style Text Processing

## Future Improvements
- Semantic Skill Matching
- OCR for Scanned Resumes
- Machine Learning-based ATS Scoring
- Spring Boot REST APIs
